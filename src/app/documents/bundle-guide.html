Bundle: Einführung & Benutzungsanweisung Installation
  1. Erstelle in deinem Homeverzeichnis einen neuen Ordner, z. B. H:\Dokumente\Cranberri.
  2. Navigiere im Dateiexplorer zu T:\...\Cranberri (im Tauschordner) und kopiere die beiden Verknüpfung-Dateien Start Cranberri, Start Minecraft und Start VS Code in den neu erstellten Ordner in deinem Homeverzeichnis.
Cranberri starten
  1. Doppelklicke auf die Verknüpfung Start Cranberri.
    ! Achtung: Es dauert kurz, bis Cranberri startet. Versuche nicht, mehrfach zu klicken.
  2. Wähle links die Welt aus, die du starten möchtest, oder klicke auf "+ New World".
  3. Klicke oben auf “Start Server”, um Cranberri zu starten. Das Konsolenfenster, das sich öffnet, ist dein Server, schließe es nicht.
  4. Du kannst Welten außerdem benennen, indem du sie auswählst und auf "Rename" klickst, danach "Confirm".
Minecraft starten
  1. Doppelklicke auf die Verknüpfung Start Minecraft.
    ! Achtung: Es dauert kurz, bis Minecraft startet. Versuche nicht, mehrfach zu klicken.
  2. Klicke, sobald Minecraft sich vollständig geöffnet hat, auf “Mehrspieler” und doppelklicke dann auf “Cranberri”, um die Welt zu öffnen.
Projekte erstellen und bearbeiten
  1. Klicke, um ein Projekt zu erstellen, im Cranberri GUI neben "Projects" auf "Create". Gib deinem Projekt einen Namen (keine Leerzeichen) und wähle über die Umschaltbuttons "simple" und Java (). "Create" zum Abschließen.
  2. Doppelklicke auf die Verknüpfung Start VS Code.
    ! Achtung: Es dauert kurz, bis VS Code startet. Versuche nicht, mehrfach zu klicken. Sobald VS Code erscheint, kannst du das Konsolenfenster schließen. Installiere keine Extensions in VS Code.
  3. Klicke nun links auf den Namen deines Projekts, um den Ordner zu öffnen. Hier findest du den Ordner scripts, in dem du ein Beispielprogramm script.java findest. Doppelklicke darauf, um es zu öffnen.
  4. Fahre mit dem nächsten Kapitel fort, wenn dein Programm fertig ist.
Cranberri-Computer platzieren und ausführen

Es ist vorausgesetzt: Du hast ein Projekt erstellt.

  1. Öffne die Cranberri-Welt in Minecraft.
  2. Erhalte einen Cranberri-Computer, indem du T drückst und den Befehl /cb ausführst. Platziere ihn auf einem passenden Block, z. B. Grasblock.
  3. Rechtsklicke den Computer, klicke dann auf das Buch ("Script..."). Gib dann das Skript an, das du ausführen möchtest: <Projektname>:<Skriptname>, z. B. my_project:script. Klicke das Buch rechts ("Confirm"), um den Dialog zu schließen, und drücke anschließend Esc.
  4. Platziere an der Seite des Computers eine Reihe von Glasblöcken in eine Richtung. Das ist die Basis für alle Pins am Computer.
  5. Platziere für jeden Pin einen Redstone-Repeater an einen der Glasblöcke. Die Pins sind vom Computer ausgehend "1" und aufsteigend nummeriert. Platziere Input-Pins so, dass das Ende des Repeaters, an dem sich zwei Redstone-Fackeln befinden, in den Glasblock zeigt. Bei Output-Pins zeigt der Repeater aus dem Glasblock heraus.
  6. Schließe Redstone-Komponenten an die Pins an.
    -> Baue bspw. die abgebildete Schaltung, wenn du das generierte Einstiegsprojekt/-skript generiert hast.
  7. Rechtsklicke erneut auf den Computer und klicke dann auf die noch weißen Wolle ("Off"), um den Computer zu aktivieren und das Skript zu starten. Die Wolle ist nun grün und zeigt "On". Verlasse den Dialog (Esc).
  8. Nach wenigen Sekunden ist der Computer einsatzbereit und im Chat ist "Starting" zu lesen.
